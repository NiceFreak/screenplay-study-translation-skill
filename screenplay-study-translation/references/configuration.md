# Configuration

Project configuration should live in `project.yaml`. Keep it explicit and portable.

## Minimal Shape

```yaml
project:
  title: Example Film
  source_language: en
  target_language: zh-CN

inputs:
  screenplay_pdf: path/to/screenplay.pdf
  subtitles: null

outputs:
  work_dir: work
  dist_dir: dist
  marker_inventory: work/source-markers.json
  subtitles_json: work/subtitles.json
  html: dist/screenplay-study.html
  pdf: null

page_mapping:
  title_pages: 1
  first_screenplay_pdf_page: 1
  displayed_page_offset: -1

translation:
  batch_size_pages: 10
  subtitle_labels: auto

audit:
  require_subtitles: false
  require_structured_markers: true
  paper_size: A4
  fail_on_missing_required_files: true
```

## Reference Subtitles

`inputs.subtitles` may be `null`. When subtitles are absent:

- do not emit subtitle labels
- skip subtitle alignment checks
- still translate with professional screenplay and subtitle judgment

## Page Mapping

Keep page mapping explicit. Do not hide title-page offsets in code.

`displayed_page_offset` is added to the physical PDF page number to compute the displayed screenplay page when no richer mapping is available.

## Audit Configuration

The audit should fail missing required files, but warn rather than fail optional subtitle absence unless `require_subtitles` is true.

`outputs.marker_inventory` should point to `source-markers.json`, generated by `scan_markers.py`.

`outputs.subtitles_json` should point to the normalized subtitle events generated by `parse_subtitles.py` when `inputs.subtitles` is configured.

`outputs.html` is the default reader/review output. `outputs.pdf` should normally be `null` in v0.1. If PDF output is deliberately reopened later, it should be a reflowed reading/print edition with source labels such as "原版第 X 页", not a page-aligned comparison PDF.

`audit.require_structured_markers` should stay true for new projects. It requires generated HTML to mark screenplay-body structure with machine-readable marker attributes, not only translated text.
