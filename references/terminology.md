## Project Terminology Template

This global reference is a template and rule reminder, not a storage location for
real-project terminology.

Project-specific terms must be written to the generated project's local
`references/terminology.md`, not to this skill-level reference file.

Terms established for a project must be applied consistently across all batches
and sessions in that project.

When subtitles are provided, dialogue translations use subtitle content
directly. Proper noun translations extracted from subtitles take priority over
AI judgment and must be populated during style profiling stage before
translation begins.

When no reference subtitles are provided:
- AI must generate a project-local terminology baseline before the first batch
- The baseline covers character names, place names, cultural concepts, and
  recurring screenplay/project terms
- The baseline does not require routine user confirmation
- Escalate to the user only for high-impact ambiguity, conflicting evidence, or
  terms that affect the user-supplied Chinese film title
- All subsequent batches must follow the generated baseline and record
  incremental additions only when genuinely new recurring terms appear

Do not commit film-specific, author-specific, or one-project terminology to this
global reference.

## General Screenplay And Format Terms

This table is a translation glossary, not a scanner or schema definition. For
entry-type names and inline markup roles, see `references/batch_schema.md`. For
marker taxonomy and scanner coverage, see `references/marker_inventory.md`.

| English | Chinese | Notes |
|---------|---------|-------|
| INT. | 内景 | Interior scene heading prefix. |
| EXT. | 外景 | Exterior scene heading prefix. |
| INT./EXT. | 内/外景 | Scene crosses interior and exterior space. |
| V.O. | 旁白 / 画外叙述 | Voice-over; usually not from a visible speaker in the scene. |
| O.S. | 离画 | Off-screen; speaker is in the scene space but outside the frame. |
| O.C. | 离镜 | Off-camera; close to off-screen, emphasizing camera frame. |
| CONT'D | 续 | Same speaker continues across dialogue blocks or pages. |
| CONTINUED | 续 | Long-form continuation marker. |
| MORE | 下页续 | Continuation marker, usually at page bottom. |
| OMITTED | 本场删去 / 删场 | Source explicitly preserves a removed scene or omitted content. |
| CUT TO | 切至 | Direct edit transition. |
| FADE IN | 淡入 | Opening or image transition. |
| FADE OUT | 淡出 | Closing or image transition. |
| DISSOLVE TO | 叠化至 | Dissolve transition. |
| SMASH CUT | 猛切至 | Abrupt transition. |
| MATCH CUT | 匹配剪接至 | Visual or conceptual match transition. |
| BACK TO | 回到 | Return transition. |
| JUMP CUT | 跳切至 | Jump cut transition. |
| CHYRON | 屏幕字幕 / 字幕卡 | On-screen location, time, or identity text. |
| TITLE ON SCREEN | 屏幕字幕 / 屏幕标题 | Text explicitly displayed on screen. |
| INSERT SHOT | 插入镜头 | Shot emphasizing an object, text, screen, or detail. |
